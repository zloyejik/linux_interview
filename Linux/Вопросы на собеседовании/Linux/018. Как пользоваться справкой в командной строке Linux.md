В Linux существует несколько встроенных способов получить справку по командам. Вот основные из них, от самых простых к более продвинутым:

### 🔹 1. `--help` или `-h` (быстрая справка)
```bash
ls --help
cp -h
```
* **Что показывает:** краткий синтаксис и список доступных опций.
* **Когда использовать:** когда нужно быстро вспомнить, какие флаги поддерживает команда.
* **Нюанс:** не все программы поддерживают `-h`, но `--help` есть почти у всех GNU-утилит.

### 🔹 2. `man` (manual pages)
```bash
man ls
man 5 passwd
man ssh_config
```
* **Что показывает:** полное руководство с описанием всех параметров, примерами, файлами конфигурации и кодами возврата.
* **Навигация внутри `man`:**
  * `Пробел` / `↓` / `PageDown` → вниз
  * `b` / `↑` / `PageUp` → вверх
  * `/текст` → поиск по тексту (регистрозависимый)
  * `n` / `N` → следующий / предыдущий результат поиска
  * `q` → выход
* **Разделы man:** одни и те же имена могут встречаться в разных разделах. Например:
  * `man 1 ls` → команда пользователя
  * `man 5 passwd` → формат файла
  * `man 8 fdisk` → административная утилита

### 🔹 3. `info` (альтернатива man)
```bash
info coreutils
info bash
```
* **Что показывает:** часто более структурированное и подробное руководство, особенно для утилит GNU.
* **Навигация:** использует гиперссылки. `Enter` → переход по ссылке, `u` → вверх по иерархии, `p`/`n` → предыдущий/следующий узел, `q` → выход.

### 🔹 4. `help` (для встроенных команд оболочки)
```bash
help cd
help export
help alias
```
* **Что показывает:** справку по командам, встроенным в сам shell (bash, zsh и т.д.).
* **Важно:** `man cd` обычно перенаправляет на `help cd`, потому что `cd` не является отдельной программой, а встроена в оболочку.

### 🔍 5. Поиск команд и описаний
* `apropos <ключевое слово>` или `man -k <ключевое слово>` → ищет команды по описанию.
  ```bash
  apropos network
  man -k "file permissions"
  ```
* `whatis <команда>` → показывает однострочное описание из man-страниц.
  ```bash
  whatis grep
  ```

### 🛠 6. Современные и сторонние инструменты
* **`tldr <команда>`** → упрощённые примеры использования (сообщественные шпаргалки).  
  Установка: `sudo apt install tldr` (Debian/Ubuntu) или `pip install tldr`.
* **`explainshell.com`** → онлайн-сервис, который разбирает команду по частям и объясняет каждый флаг.
* **`cheat <команда>`** → аналог tldr, часто предустановлен в некоторых дистрибутивах.

### 💡 Практические советы
1. Начинайте с `--help`, если нужно быстро понять опции.
2. Если нужны детали, примеры или описание форматов файлов → `man`.
3. Если команда не найдена в `man`, проверьте, установлен ли соответствующий пакет.
4. Для поиска неизвестной команды по задаче используйте `apropos` или `man -k`.
5. В `man` можно сразу открыть нужный раздел: `man 1 intro`, `man 7 signal`.

Если скажете, с какой именно командой или задачей столкнулись, помогу подобрать оптимальный способ получения справки.
